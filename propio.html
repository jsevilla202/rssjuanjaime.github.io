¿Cómo creo mi propio RSS?

Crear un RSS es como crear un XML y enlazarlo con una página web

A continuación, hay un ejemplo de un código de RSS que se podría implantar en una web

Click Aquí

Podemos diferenciar varias etiquetas:

  • rss: La etiqueta de la RSS
  • channel: Canal de la RSS
  • title: Titulo de la noticia
  • link: Link a la noticia
  • description: Descripción de la noticia
  • item: Contenido de la noticia

La primera línea del documento (la declaracion de XML) define la version de XML y la codificación de caracteres usados en el documento. En este caso el documento usa la versión 1.0 de XML y el set de caracteres UTF8.

La siguiente línea es la declaracion del RSS que identifica que el documento es de tipo RSS (en este caso 2.0).

La siguiente línea contiene el elemento channel , usado para describir con un titulo el feed del RSS.

El elemento channel requiere 3 elementos hijo (y en nuestro caso un cuarto opcional):

  • title Define el título del canal. (El podcast de Radios Libres)
  • link Define el hyperlink al canal. (https://www.radioslibres.net/)
  • description Describe el canal (Radios Libres presenta su podcast sobre radios comunitarias y tecnologías libres.)
  • copyright Muestra el copyright

Cada elemento channel puede tener 1 o mas elementos item .

Cada elemento item define un articulo o historia en el feed del RSS.

El elemento item tiene 3 elementos hijo.

El elemento item requiere 3 elementos hijo (y en nuestro caso algunos opcionales):

  • title Define el título del item. (Episodio 1 – Bienvenida)
  • link Define el hyperlink al item. (https://www.radioslibres.net/podcasts/episodio-1.html)
  • description Describe el item(En el episodio inaugural de nuestro primer podcast les contaremos cómo será esta temporada.)
  • author Muestra el autor del articulo (Radios Libres).
  • pubDate Muestra la fecha de publicación de la noticia o articulo ( Thu, 12 Sep 2019).
  • enclosure Contiene link de archivo usado en la noticia o articulo (url=»https://www.radioslibres.net/podcasts/episodio1.ogg» type=»audio/ogg» length=»300″ /)

Finalmente, las dos ultimas líneas cierran los elementos channel y rss .
